# 폰트 병합 기능

2개의 폰트를 병합하는 기능을 만들고 싶어. 예를 들어 A 폰트에 없는 글리프를 B폰트에서 가져올 수 있는 기능이야.

*   2개의 폰트를 선택한 후 마우스 우클릭 "폰트 병합" 메뉴 선택시, 폰트 병합 팝업 호출. (단축키는 Ctrl(Cmd) + Shift + F로 등록)
    
*   선택한 2개의 폰트가 등록되어 있고, "A 폰트" 에 "B 폰트" 를 병합하도록 처리되어 있고, 순서 변경 버튼으로 대상 폰트를 변경할 수 있어.
    
*   병합될 폰트 이름을 지정하는 필드가 따로 존재하고, 디폴트값은 "{A폰트명}{B폰트명}Merged".ttf(또는 otf)로 설정되고, 사용자가 임의로 변경할 수 있어.
    
*   "병합" 버튼을 클릭하면, 폰트가 병합처리 후 저장하게 되.
    

# 폰트 입력 테스트 기능

폰트를 선택한후 Spacebar 클릭시 폰트 테스트 팝업이 호출되. 디폴트값으로 Random 문자열 생성

### 팝업 내 필요한 기능들

*   현재 선택된 폰트 이름 정보 출력 필드
    
*   폰트 크기 선택 드롭다운 리스트 버튼.
    
*   A버튼 : 대문자 A부터 Z까지 입력됨.
    
*   a버튼 : 소문자 a부터 z까지 입력됨.
    
*   0-9버튼 : 숫자 0부터 9까지 입력됨.
    
*   Random : 랜덤 문자열을 생성해주는 기능 (폰트의 언어 특성에 맞게 랜덤 문자열 생성)
    
*   텍스트 입력 필드 : 사용자가 원하는대로 입력하면 해당 폰트로 텍스트가 출력됨. (단일 라인이 아닌 멀티 라인 가능. 사용자 입력에 따라 자동으로 늘어나지만 최대 5라인 이상일 경우 스크롤 처리)
    
*   프리뷰 필드 : 텍스트가 나열되는 공간.
