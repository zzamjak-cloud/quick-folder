# 이미지 크롭(영역 추출) 기능 설계

## 개요

이미지 미리보기 모달에서 사각형 영역을 드래그 선택한 후, 해당 부분만 별도 PNG 파일로 추출하는 기능.

## 대상

- JPG, PNG 이미지만 (PSD, 동영상, 텍스트 등 제외)
- 모달 미리보기(PreviewModals.tsx)에서만 제공

## 인터랙션

### 영역 선택
- 모달이 열리면 이미지 위에서 **바로 드래그**하여 사각형 영역 선택
- 별도 크롭 모드 진입 없음 (항상 드래그 가능)

### 선택 영역 표시
- 선택 영역 바깥: 반투명 어두운 오버레이
- 선택 영역 테두리: 녹색 실선 (`#4ade80`)
- 4개 모서리에 핸들 표시
- 선택 영역 하단에 실제 픽셀 크기 표시 (예: `640 × 480 px`)

### 조작
- **드래그**: 새 선택 영역 생성 (기존 선택 대체)
- **모서리 핸들 드래그**: 크기 조절
- **영역 안 드래그**: 선택 영역 이동
- **Shift + 드래그**: 1:1 정사각형 비율 고정
- **빈 영역 재드래그**: 이전 선택 대체

### 저장
- 선택 영역이 있을 때 "PNG 저장" 버튼 표시
- 클릭 시 Rust 백엔드 `crop_image` 커맨드 호출

## 저장 규칙

- **위치**: 원본 파일과 같은 폴더
- **파일명**: `{stem}_crop.png`
- **중복**: `{stem}_crop(2).png` (helpers.rs `find_unique_path` 활용)
- **포맷**: 항상 PNG

## 구현 범위

### 프론트엔드 (PreviewModals.tsx)

1. 이미지 모달에 Canvas 오버레이 레이어 추가
2. 마우스 이벤트 처리:
   - `mousedown` → 드래그 시작 (새 선택 or 핸들 리사이즈 or 영역 이동 판별)
   - `mousemove` → 선택 영역 업데이트 + Canvas 리렌더
   - `mouseup` → 드래그 종료
3. Canvas 렌더링:
   - 어두운 오버레이 (선택 영역 바깥)
   - 녹색 테두리 + 모서리 핸들
   - 픽셀 크기 라벨
4. 화면 좌표 → 원본 이미지 픽셀 좌표 변환 (이미지 표시 크기 vs 원본 크기 비율)
5. Shift 키 비율 고정 로직
6. "PNG 저장" 버튼 (선택 영역 존재 시만 표시)
7. 저장 후 토스트 알림 + 파일 목록 새로고침

### Rust 백엔드 (lib.rs)

1. `crop_image` 커맨드 추가:
   - 입력: `path: String, x: u32, y: u32, width: u32, height: u32`
   - `image` crate로 원본 로드 → `crop_imm()` → PNG 저장
   - `helpers::find_unique_path()`로 중복 파일명 회피
   - 출력: 저장된 파일 경로 반환

## 비범위 (미포함)

- Undo 지원 (원본 파일이 보존되므로 불필요 — 크롭 파일 삭제로 충분)
- 컬럼 뷰 사이드 패널에서의 크롭
- 비율 프리셋 (16:9, 4:3 등)
- 크롭 미리보기 (별도 모달)
- PSD/동영상 크롭
