PNG 또는 JPG 파일 선택후 Enter 클릭시 간단한 이미지 편집 기능을 넣고 싶어. 주로 이미지를 누군가에게 전달할때, 이미지에 주요 설명을 추가해서 전달하고 싶을때 사용하기 위한 기능.

1.  이미지 Crop 기능 : Rect를 사용자가 직접 드래그해서 이미지를 잘라내는 기능. 포토샵의 Crop과 같은 동작. Rect의 상하좌우 핸들러를 이동시켜서 조절 가능. 조절 후 적용 버튼 클릭시 편집상태 적용처리
    
2.  이미지 위에 도형 아웃라인을 그리는 기능. 편집 과정에서는 벡터 형태로 크기 및 위치 조정 가능. 아웃라인 컬러 및 두께 조절 기능 필요.
    
3.  이미지 위에 텍스트를 입력하는 기능.
    
4.  이미지 위에 드로잉 기능. 타블렛 펜 또는 마우스로 직접 드래그해서 그리는 펜 기능. 펜 두께 및 컬러 조절 기능 필요.
    
5.  요소 선택 및 이동 기능
    
6.  지우개 기능 : 도형 제거 및 펜 드로잉 제거 기능
    
7.  초기화 버튼 : 원본 이미지로 되돌리기 기능.
    
8.  레이어 기능 : 이미지 위에 도형 아웃라인, 텍스트, 펜 드로잉 등을 레이어로 나눠서 적용할 수 있는 기능 (포토샵과 동일한 형태)
    
9.  저장 버튼 : 저장하기 클릭시 "{파일명}_Desc".확장자 로 별도 저장.
    
10.  이미지 편집 에디터에는 간단한 툴바와 레이어 패널, 작업 패널, 헤더 프리셋 패널로 구성해줘.
