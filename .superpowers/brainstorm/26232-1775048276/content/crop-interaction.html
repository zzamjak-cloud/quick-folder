크롭 모드 진입 & 저장 방식

모달 미리보기에서 크롭 기능을 어떻게 사용하게 할까요?

A

항상 드래그 가능 (바로 선택)

모달이 열리면 바로 이미지 위에서 드래그하여 사각형 영역 선택 가능. 선택하면 "저장" 버튼 표시.

파일명.jpg
이미지 영역
💾 저장
✅ 간단 · ⚠️ 실수로 선택될 수 있음
B

크롭 버튼 클릭 → 크롭 모드 진입

모달 헤더에 "크롭" 버튼 추가. 클릭하면 크롭 모드 진입 → 드래그로 영역 선택 → 저장/취소 버튼 표시.

✂️ 크롭 ✕
이미지 영역
취소 저장
✅ 명시적 진입 · ✅ 실수 방지 · 기존 패턴(픽셀화 등)과 일관
