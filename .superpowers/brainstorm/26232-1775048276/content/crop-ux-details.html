PNG 저장
• 바깥 영역은 반투명 어둡게 처리
• 선택 영역 테두리: 녹색 점선/실선
• 모서리 핸들로 크기 조절 가능
• 선택 영역 안을 드래그하면 이동
• 실제 픽셀 크기(px) 표시

추가 기능 — 필요한 것만 골라주세요

1

비율 고정 (Shift 드래그)

Shift 키를 누르고 드래그하면 정사각형(1:1) 비율 유지

2

ESC로 선택 취소

선택 영역이 있을 때 ESC → 선택 해제. 선택 없을 때 ESC → 모달 닫기.

3

재선택 (빈 영역 다시 드래그)

이미 선택이 있어도 빈 영역에서 새로 드래그하면 이전 선택 대체

4

기본 동작만 (추가 기능 없음)

드래그 선택 + 저장 버튼만. 심플하게.
