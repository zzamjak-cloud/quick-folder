이미지 크롭 기능 — 최종 디자인

확정된 설계를 정리했습니다. 확인 후 승인해주세요.

1. 위치 & 진입

• 모달 미리보기(PreviewModals.tsx)에서만 제공
• 별도 크롭 모드 없이, 이미지 위에서 바로 드래그하여 영역 선택
• JPG, PNG 이미지에만 적용 (PSD, 동영상, 텍스트 제외)

2. 선택 영역 인터랙션

640 × 480 px
PNG 저장
🖱 드래그 — 새 선택 영역 생성 (기존 선택 대체)
↔️ 모서리 핸들 드래그 — 크기 조절
✋ 영역 안 드래그 — 선택 영역 이동
⇧ Shift + 드래그 — 1:1 정사각형 비율 고정

3. 저장

• "PNG 저장" 버튼 클릭 시 Rust 백엔드에서 크롭 실행
• 저장 위치: 원본 파일과 같은 폴더
• 파일명: {stem}_crop.png
• 중복 시: {stem}_crop(2).png (helpers.rs 패턴)
• 저장 후 토스트 알림 + 파일 목록 새로고침

4. 구현 범위

프론트엔드
PreviewModals.tsx에 크롭 UI 추가
Canvas 기반 선택 영역 렌더링
마우스 이벤트 (드래그, 리사이즈, 이동)
Shift 비율 고정 로직
좌표 → 원본 이미지 픽셀 변환
Rust 백엔드
crop_image 커맨드 추가
입력: path, x, y, width, height
출력: {stem}_crop.png 저장
image crate로 크롭 처리
helpers::find_unique_path 활용

5. Undo 지원

• UndoAction에 crop 변형 추가 (= create_file과 동일하게 삭제로 복원)
• Ctrl+Z 시 생성된 크롭 파일을 휴지통으로 이동
